## 🚀 使用方式

### 快速开始
```bash
# 1. 安装依赖
pip install requests

# 2. 配置API密钥
export OPENAI_API_KEY="your-key"

# 3. 运行程序
python main.py
```

### 命令行使用
```bash
# 生成脚本
python main.py generate "主题" --template popular_science

# 查看模板
python main.py templates
```
